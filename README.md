FC quest project
============================

REQUIREMENTS
------------
 PHP 5.6.0.
 Mysql


INSTALLATION
------------
~~~
git clone git@bitbucket.org:jackshadow/quest.git
cd /quest
composer global require "fxp/composer-asset-plugin:~1.0.3"
composer install
~~~
Создайте базу данных в Mysql
Поменяйте настройки в файле
config/db.php
